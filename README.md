Enkelt textbaserad RPG spel gjort i C# med microsoft visual studio som slutprojekt i programmering 1 på gymnasiet. 
Du dräper spöken och samlar på dig guld som du kan köpa saker med för att befria en by från en förbannelse.

(Simple text-based RPG game made in C# with Microsoft Visual Studio as a final project in Programming 1 in high school. 
You kill ghosts and collect gold that you can use to buy things to free a village from a curse.)
